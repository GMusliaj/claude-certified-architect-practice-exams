P1.3 Weak-area drill mode

"Drill weak areas" button on results screen (appears only when any domain scored < 75%). Launches a 10-question mini-exam drawing exclusively from domains where the user underperformed. Reuses buildExam() with a filtered bank. Acceptance: questions sourced only from flagged domains; shows which domains were targeted in the start screen.
