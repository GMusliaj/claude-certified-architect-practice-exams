## Purpose

This repository was built to demonstrate practical usage of the Claude ecosystem —
specifically how to combine **Claude Code hooks**, **Model Context Protocol (MCP)
tool design**, **agentic pipelines**, and **structured prompt engineering** into a
cohesive local application. The exam questions and practice material are derived
from publicly available Anthropic documentation and are intended solely to support
self-directed study.
